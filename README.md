# TokenRun

Website: **[token-run.com](https://token-run.com)**

This repository is the official channel for collecting **issue reports and suggestions** from TokenRun users.

Source code is not published here — this repo is dedicated to **bug reports, feature requests, and general feedback**.

## How to Participate

### 1. Report an Issue / Suggest an Idea

Go to the **[Issues](../../issues)** tab and click **New issue**, then pick one of the templates below:

- 🐛 **Bug Report** — Report errors or unexpected behavior in the app
- 💡 **Feature Request** — Suggest new features or improvements
- 💬 **General Feedback** — Ask questions or share any other feedback

### 2. When Submitting

- Please describe **reproducible, concrete situations** whenever possible.
- Screenshots or screen recordings help us understand the problem much faster.
- Search existing issues first — duplicates may be merged into the original issue.
- Do **not** include personal information (email, phone number, payment details, etc.) in public issues.

## How Issues Are Handled

- Every submitted issue is reviewed and labeled.
- Progress is tracked via labels such as `triage`, `accepted`, `in-progress`, `resolved`.
- Turnaround time may vary depending on severity and priority — thank you for your patience.

Your feedback makes TokenRun better. Thank you.
